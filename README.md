HackTheNorth
============

Our Myo hack for the HackTheNorth Hackathon September 19th 2014

##Requirements##
In order to build, you first need to add these Standard Assets. Click on `Assets>Import Package>` for the following Assets:
	
- Character Control
- Scripts
- Skyboxes
- Terrain
